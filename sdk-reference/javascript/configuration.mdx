#### What you will learn
- Configuring the infrasack.ai Javascript SDK for your needs

#### Prerequisites
- An application using the infrasack.ai Javacript SDK `@infrastack/otel`

### Overview
Our Javascript SDK provides flexibility to developers to configure the SDK. There are 3 ways to configure the SDK:
- Via environment variables
- Via code
